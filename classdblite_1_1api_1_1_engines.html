Returns an opened Database, the first created if opts is `None`.

Parameters
optsdatabase connection options, engine-specific; SQLite takes a file path or path-like object or `":memory:"`, Postgres takes a Postgres URI scheme like `"postgresql://user@localhost/mydb"` or a Postgres keyword=value format like `"host=localhost username=user dbname=mydb"` or a dictionary of connection options like `dict(host="localhost", ..)`
enginedatabase engine if not auto-detecting from connection options, "sqlite" for SQLite3 and "postgres" for PostgreSQL (case-insensitive)
kwargsadditional arguments given to engine constructor, e.g. `detect_types=sqlite3.PARSE_COLNAMES` for SQLite, or `minconn=1, maxconn=4` for Postgres connection pool
